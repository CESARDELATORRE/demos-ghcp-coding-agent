# Demo Project Example

This directory demonstrates the structure for creating demo projects in this repository.

## Creating New Demos

1. Create a new subdirectory for your demo
2. Follow the instructions in `.github/copilot-instructions.md`
3. Include a README.md explaining the specific demo
4. Test thoroughly before committing

## Current Demo

This is a placeholder for future demo projects. The instructions file contains examples of how to create:
- .NET console applications
- ASP.NET Core Web APIs
- Node.js applications  
- Python scripts
- Full-stack demos

Refer to `.github/copilot-instructions.md` for complete guidance.
